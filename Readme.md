# Luxury Hotel Analytics Dashboard ✨

An executive-style interactive hotel analytics dashboard built using AI-assisted development, modern web design principles, and data storytelling techniques.

## 🚀 Project Overview
This project combines:
- AI-assisted dashboard development
- Modern SaaS-inspired UI/UX
- Interactive HTML analytics experience
- Executive-level storytelling visuals

The dashboard was created using:
- **ChatGPT** → KPI logic, storytelling, analytical flow
- **Claude AI** → HTML/CSS refinement and UI structuring
- **Dribbble Inspiration** → Layout and premium design concepts

## 📊 Features
- Multi-page analytics dashboard
- Executive KPI cards
- Revenue & booking analysis
- Guest insights
- Operational analytics
- AI-generated insight sections
- Interactive charts using Chart.js
- Responsive modern UI


However, human intervention was still essential for:
- Business understanding
- Design judgment
- Analytical thinking
- UX improvements
- Final refinements
